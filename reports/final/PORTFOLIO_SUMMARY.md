# 권장과목 기반 학과 군집화와 전문가 검증 — 포트폴리오 요약

> 입시 상담 현업 문제를 데이터화하고, 비지도 군집화로 구조를 도출한 뒤,
> 현직 전문가 14명의 카드소팅으로 **외부 검증**하고, 그 결과를 상담 워크플로우로 연결한
> 탐색적 데이터 분석 프로젝트.

## 문제 정의

입시 상담에서 "어떤 학과들이 권장과목 관점에서 서로 비슷한가"는 컨설턴트의 경험적 직관에
의존한다. 이 직관을 **고등학교 권장과목 데이터만으로** 재현·보완할 수 있는지, 그리고 그 결과가
실제 전문가 판단과 부합하는지를 검증한다. 목표는 완성된 추천 시스템이 아니라, 상담을 보조하는
신호를 찾는 탐색적 분석이다.

## 데이터

- 울산 지역 산업 맥락을 반영한 **25개 학과** (목적 표본, 통계적 대표 표본 아님).
- 학과 과목 선택 가이드의 일반/진로/융합 선택과목 → **25×89 권장과목 binary matrix**.
- 외부 검증: 현직 입시 컨설턴트 **14명**의 open card sorting → 25×25 동시분류 행렬
  (개인정보는 집계 단위로만 사용, 원자료 비공개).

## 방법론

- **유사도**: IDF 가중 cosine similarity. 공통 과목(예: 확률과 통계, 21/25 학과)의 변별력 저하를
  파라미터 없는 IDF(`w = ln(N/df)`)로 보정. 가중 강도는 단일 조절 모수 α 민감도 분석으로 검증.
- **군집화**: average linkage hierarchical clustering(주) + k-means(비교).
- **외부 검증**: 카드소팅 consensus clustering과 알고리즘 군집을 ARI로 비교.
- **불확실성 정량화**:
  - rater 부트스트랩(B=1000)으로 consensus 안정성(Jaccard)과 ARI 신뢰구간,
  - **쌍체 부트스트랩 차이**(ARI_IDF − ARI_binary)로 IDF 우위의 견고성,
  - feature(과목) 부트스트랩으로 코어 하위 군집 안정성.
  - 모든 부트스트랩 seed 고정(재현 가능).

## 주요 결과

- **거시 구조(k=3)는 전문가 합의와 방향상 일치**: ARI 점추정 0.84. 단 14명 소패널이라
  부트스트랩 CI가 [0.04, 0.95]로 넓어 *크기*는 단정하지 않고 방향성으로 해석.
- **IDF 가중의 이점은 상위 분할이 아니라 세분 구조에 제한적**:
  쌍체 부트스트랩 기준 **k=6에서만** binary 대비 우위가 비교적 견고(차이 CI [0.06, 0.32], 0 제외),
  k=5는 경향 수준(CI 0 포함), k=4는 오히려 binary가 근소 우위 — 모두 "상위 구조는 STEM-보건
  대군집이 지배"라는 해석과 정합.
- **코어 하위 군집(의약-보건/정량·응용/공학-화학) 안정성은 모두 중간 수준**(Jaccard ~0.56–0.58).
  특정 가지가 유달리 견고하지는 않음(이전 초안의 수기값 0.97 등은 재현 가능한 추정이 아니어서 폐기).
- **내부-외부 불일치를 두 유형으로 구조화**:
  - Type 1(알고리즘 O / 전문가 X): 준비기반은 유사하나 전문가가 분리 (예: 신소재-생명).
  - Type 2(알고리즘 X / 전문가 O): 전문가는 함께 추천하나 권장과목이 분기 (예: 조선해양-자동차).

## 실무 적용

알고리즘×전문가 일치/불일치 **2×2 사분면**을 상담 워크플로우로 연결:

| 유형 | 의미 | 상담 활용 |
|---|---|---|
| 알고리즘 O / 전문가 O | 안정 추천군 | 우선 추천 단위 |
| 알고리즘 O / 전문가 X (Type 1) | 전문가가 놓칠 수 있는 준비기반 유사 | 교차 계열 대안 제시 |
| 알고리즘 X / 전문가 O (Type 2) | 직관상 함께지만 권장과목 분기 | 과목 정합성 경고, 보강 안내 |
| 알고리즘 X / 전문가 X | 준비·직관 모두 낮은 관련성 | 추천 후순위 |

→ 학생의 실제 이수 과목과 학과별 권장과목 프로파일을 대조하는 의사결정 보조 도구로 확장 가능.

## 한계

1. **표본**: 25개 학과 목적 표본 → 전체 학과 일반화 한계.
2. **권장과목 해상도**: 수학·과학 공통 과목이 많아 STEM 내부 세부 차이 반영 부족.
3. **전문가 패널**: 14명 소패널 → ARI CI가 넓어 일치도 크기 단정 곤란(방향성 위주 해석).
4. **성과 검증**: 실제 입결·합격·학생 선택 결과와 미연결.

## 후속 개선 계획

- 입결 feasibility를 결합한 **Type 2 "준비 정합성 위험"의 정량화**.
- 25개 학과를 anchor로 한 **전 학과 준지도 확장**(신규 학과를 가장 가까운 anchor 군집에 투영,
  경계 사례만 선택적 전문가 검증).

## 기술 스택

Python (NumPy, pandas, scipy, scikit-learn, matplotlib), LaTeX(IEEEtran), Git.
재현 가능한 파이프라인: 행렬 구성 → IDF 군집화 → 카드소팅 집계 → 부트스트랩 검증 → 그림/표 생성.
