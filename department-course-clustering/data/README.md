# Data Directory

이 폴더는 데이터의 공개 가능성과 처리 단계를 기준으로 나눕니다.

## raw

원본 자료를 보관합니다. 컨설턴트 개별 응답, 본사 입결 데이터, 원본 엑셀 파일 등은 이곳에 둡니다. 기본적으로 GitHub에 올리지 않습니다.

## interim

정제 중간 산출물을 보관합니다. 예를 들어 학과명 표준화 중간 파일, 대학별 모집단위 매칭 파일 등이 여기에 해당합니다.

## processed

분석 코드가 바로 읽는 최종 가공 데이터를 보관합니다. 공개 저장소에 올릴 경우 개인 또는 기관 식별 정보와 민감한 입결 원자료가 제거되어야 합니다.

## external

대교협, 커리어넷, 대학어디가 등 외부 공개자료를 보관합니다. 라이선스나 재배포 가능 여부가 불확실한 파일은 원본을 올리지 말고 출처와 접근일만 기록합니다.
